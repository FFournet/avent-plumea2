Les plateformes d’écriture
par @harleyawarren

Aujourd’hui, je vous propose une case un peu spéciale. Vous aimeriez publier vos écrits en ligne mais vous ne savez pas où ? Ce guide est fait pour vous.

Il existe de nombreuses plateformes où un auteur peut publier ses écrits, mais toutes ne se ressemblent pas et toutes ne vous correspondront pas forcément. Bien choisir votre plateforme vous permettra de maximiser vos chances de trouver votre public.

Wattpad
Lien vers le site
Les avantages :

• La plus grande communauté d’écriture au monde
• Intéresse les éditeurs traditionnels

Les inconvénients :

• Public très jeune
• Difficile de se démarquer dans la masse


Inkitt
Lien vers le site
Les avantages :
• Moins de monde que sur Wattpad, il est plus facile d’être remarqué
• Possibilité de se faire éditer via leur plateforme Galatea

Les inconvénients :
• Très axé romance
• Communauté FR peu développée, il vaut mieux y écrire en anglais


L’Atelier des Auteurs
Lien vers le site
Les avantages :
• Cours d’écriture (payants) intégrés à la plateforme
• Défis d’écriture pour s’entraîner

Les inconvénients :
• Niveau souvent assez faible
• On refusera de vous lire si votre texte fait plus de 1500 mots


Plume d’Argent
Lien vers le site
Les avantages :

• Communauté sympathique et animée
• Un système un peu comme celui de Pluméa, où il faut s’intéresser aux textes des autres avant de publier le sien

Les inconvénients :

• Pas du tout adapté pour les textes adultes


Fyctia
Lien vers le site
On passe sur une plate-forme un peu différente, plus « professionnelle », au fonctionnement particulier.

Les avantages :

• Possibilité d’être édité si l’on gagne le concours

Les inconvénients :

• Romance uniquement
• Publication restreinte au cadre des concours


KDP Amazon

Lien vers le site

Les avantages :

• Présence dans le catalogue Kindle
• La plus rentable financièrement parlant
Relativement simple d’utilisation

Les inconvénients :

• Il s’agit d’une publication « officielle » en auto-édition, il peut donc être difficile de proposer votre texte à un éditeur par la suite
• Contribue à financer les polissoirs de crâne de Jeff Bezos


Archive of our own (AO3)
Lien vers le site
Terminons par la plateforme la moins conventionnelle de la liste
Les avantages :

• Pas de limite de contenu, super pour le NSFW

Les inconvénients :

• Très majoritairement anglophone
• Plus pensé pour la fanfiction que pour les romans originaux
